# Academic Website (GitHub Pages)

This is a static personal website template designed for an academic job market profile.

## Files
- `index.html` - page structure and content
- `styles.css` - visual design and responsive layout
- `script.js` - mobile menu + footer year

## Deploy on GitHub Pages
1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Copy these files into that repository.
3. Commit and push to the `main` branch.
4. Your site will be available at `https://<your-github-username>.github.io/`.

## Customize content
Update placeholders in `index.html`:
- Name, title, department, institution
- Research statement and themes
- Publications and working papers
- Teaching experience
- Contact details and profile links

## CV link
The homepage expects a file named `cv.pdf` in the repository root.
- Add your real CV file as `cv.pdf`, or
- Change the `Download CV` button link in `index.html`.
